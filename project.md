# Student Record Management System

## 📌 Project Overview

Student Record Management System is a web-based application developed to manage student information digitally. It helps store, update, view, and organize student records efficiently.

## ✨ Features

* Student Registration
* Add Student Records
* View Student Details
* Update Student Information
* Delete Student Records
* Attendance Management
* Marks Management

## 🛠️ Technologies Used

### Frontend

* HTML
* CSS
* Bootstrap
* React

### Backend

* Java
* Spring Boot

### Database

* MySQL

## 🎯 Objective

To reduce manual record maintenance and improve efficiency in managing student data.
